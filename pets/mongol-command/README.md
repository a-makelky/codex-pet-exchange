# Mongol Command

A synthwave Mongol command warrior for Codex desktop.

Install ID:

```text
custom:mongol-command
```

Install command if the zip is in Downloads:

```bash
mkdir -p ~/.codex/pets
ditto -x -k ~/Downloads/mongol-command-codex-pet.zip ~/.codex/pets
ls -la ~/.codex/pets/mongol-command
```
